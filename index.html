Situations professionnelles

Situation 1 - Création d'un site web de portfolio

Contexte : réalisation d'un site web pour présenter mon parcours et mes compétences dans le cadre du BTS SIO.

Besoin exprimé : disposer d'un support clair, propre et structuré pour l'épreuve et pour présenter mes projets.

Solutions proposées : création d'un site en HTML, CSS et JavaScript avec plusieurs sections de présentation.

Technologies utilisées : HTML, CSS, JavaScript.

Difficultés rencontrées : organiser les informations, structurer les sections et rendre le site lisible.

Résultats obtenus : un site simple, fonctionnel et facile à modifier.

Situation 2 - Projet de site éducatif

Contexte : projet autour d'un site éducatif de jeux pour apprendre le développement, le SQL et la sécurité informatique.

Besoin exprimé : créer une plateforme motivante pour apprendre en pratiquant.

Solutions proposées : imaginer une structure de site avec des mini-jeux, des exercices et une interface simple.

Technologies utilisées : HTML, CSS, JavaScript, SQL.

Difficultés rencontrées : concevoir un projet cohérent et relier l'aspect ludique avec l'apprentissage.

Résultats obtenus : une base de projet intéressante à développer pour le BTS.
