BTS SIO SLAM

Qu’est-ce que le BTS SIO ?

Le BTS SIO forme aux métiers de l’informatique. L’option SLAM est orientée vers le développement d’applications, la programmation, les bases de données et la maintenance de solutions logicielles.

Compétences BTS travaillées

  • Gérer le patrimoine informatique
  • Répondre aux incidents et aux demandes d’assistance
  • Développer la présence en ligne d’une organisation
  • Travailler en mode projet
  • Mettre à disposition un service informatique
  • Organiser son développement professionnel
Mon école : ÉSTIAM

École d’informatique appliquée

ÉSTIAM est une école spécialisée dans l’informatique appliquée aux métiers. Le parcours proposé peut aller jusqu’à Bac+5, avec une logique progressive et professionnalisante.

BTS SIO SLAM

Le BTS SIO SLAM est une formation Bac+2 sur 24 mois. Elle prépare aux métiers liés à la conception, au développement et à la maintenance de programmes applicatifs.

Alternance

La formation peut se faire en alternance dès la première année. Cela permet de compléter les cours par une expérience professionnelle en entreprise.

08

Veille technologique

Sujet de veille

Ma veille technologique porte sur Java et le modding Minecraft. Ce sujet est lié à mon intérêt pour le développement, le jeu vidéo et la création de mods.

Problématique

Comment l’évolution de Minecraft Java Edition et de Java influence-t-elle le développement, la compatibilité et la maintenance des mods Minecraft ?

Méthode de veille

Pour que ma veille soit régulière, je m’appuie sur des sources capables d’envoyer des notifications : newsletter, flux RSS, notifications YouTube et alertes provenant de médias spécialisés. Cela évite une simple vérification manuelle et me permet de suivre les changements au fil du temps.

Journal de veille
Février 2026

Snapshots Minecraft 26.1

Suivi des snapshots pour repérer les changements techniques avant la version stable. Impact : anticiper les risques de compatibilité pour un mod.

Mars 2026

Préversions et sortie 26.1

Suivi des pre-releases puis de la sortie officielle. Impact : comprendre que les changements de datapack, resource pack ou JVM peuvent influencer les tests de mods.

Avril 2026

Hotfix et corrections

Suivi d’un correctif après sortie. Impact : une mise à jour mineure peut corriger un bug important, mais aussi demander de retester un projet.

Bilan intermédiaire

Ce que je surveille

Version Java demandée, changements techniques Minecraft, corrections après sortie, tendances de la communauté modding et impacts sur les outils de développement.

Articles récents retenus — 3 derniers mois
Article Date Information retenue et impact
Minecraft 26.1 Snapshot 6 03/02/2026 Info : changements sur les mobs bébés et formats de données de monde.
Impact : certains mods liés aux entités, mondes de test ou données peuvent devoir être retestés.
Minecraft 26.1 Snapshot 9 18/02/2026 Info : correction d’un crash lié aux entités sortant des limites verticales du monde.
Impact : les corrections de bugs peuvent changer le comportement observé en test.
InfoQ — JDK 26 and JDK 27 20/02/2026 Info : JDK 26 est présenté comme une version non-LTS avec 10 JEPs. L’article cite notamment HTTP/3 dans l’API HTTP Client, Structured Concurrency, Lazy Constants, Vector API et la suppression de l’Applet API.
Impact : même si Minecraft ne change pas de version Java à chaque sortie, l’environnement Java évolue vite. Pour un projet de modding, il faut surveiller la version du JDK utilisée par Minecraft, les outils de build et les loaders.
Minecraft 26.1 Pre-Release 1 10/03/2026 Info : préversion avec changements techniques, versions datapack/resource pack et composants.
Impact : les pre-releases aident à anticiper les adaptations avant une version stable.
Minecraft 26.1 Pre-Release 3 17/03/2026 Info : corrections de bugs et ajustements avant la sortie officielle.
Impact : je peux repérer les corrections à tester si un comportement change dans le jeu.
Minecraft Java Edition 26.1 24/03/2026 Info : sortie officielle avec contenu et éléments techniques, dont des options JVM et des changements liés à la lumière.
Impact : une version stable sert de base pour tester la compatibilité d’un mod.
Minecraft Java Edition 26.1.2 09/04/2026 Info : hotfix corrigeant des problèmes critiques après la sortie officielle.
Impact : les correctifs mineurs sont importants car ils peuvent résoudre ou modifier des problèmes rencontrés par les joueurs et moddeurs.

Ce que j’ai appris

Cette veille m’a appris qu’un mod Minecraft dépend de plusieurs évolutions : version du jeu, version Java, outils de build, loaders et correctifs. Je dois donc suivre les snapshots, pre-releases, sorties officielles et hotfixs, mais aussi les évolutions du JDK pour comprendre l’environnement technique autour du modding.
