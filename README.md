# Portfolio Enzo Bertrand

Portfolio professionnel BTS SIO SLAM.

## Structure du projet

```txt
Portfolio_Enzo_Bertrand/
├── index.html
├── assets/
│   ├── css/
│   │   └── style.css
│   ├── js/
│   │   └── script.js
│   └── images/
│       └── images, icônes et visuels du portfolio
├── docs/
│   └── CV et documents PDF
└── game/
    └── mini-jeu Debug Invaders
```

## Rôle des dossiers

- `assets/css` : contient les fichiers de style du portfolio.
- `assets/js` : contient les scripts JavaScript du portfolio.
- `assets/images` : contient les images, icônes, GIF et visuels utilisés dans le site.
- `docs` : contient les documents liés au portfolio comme le CV et les PDF de projets.
- `game` : contient le mini-jeu intégré au portfolio.

## Lancement

Ouvrir `index.html` dans un navigateur.

## Déploiement GitHub Pages

Pour GitHub Pages, déposer tout le contenu du dossier `Portfolio_Enzo_Bertrand` à la racine du dépôt.
